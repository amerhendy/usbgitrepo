# 💎 Amer Sync - Crystal Edition 2026

![Amer Sync Icon](icon.png) 

**Amer Sync** هو نظام مكتبي متطور مصمم خصيصاً للمبرمجين والمطورين (خاصة عشاق **Laravel**) لإدارة وتزامن المشاريع بين جهاز العمل والفلاشة (USB) بضغطة زر واحدة، باستخدام قوة محرك **Git**.

## ✨ الميزات الرئيسية (Features)
- **واجهة كريستالية (Crystal UI):** تصميم عصري يعتمد على نظام الألوان الموحد (Pattern System) لراحة العين.
- **إدارة بيئات العمل:** تنظيم مشاريعك في مجموعات (مثل: البيت، المكتب، عام).
- **مزامنة ذكية:** تنفيذ عمليات `Push` و `Pull` تلقائياً دون الحاجة لكتابة أوامر في الـ Terminal.
- **إدارة المشاريع:** إضافة، تعديل، وحذف المشاريع والمسارات بسهولة تامة.
- **متجاوب (Responsive):** تصميم مدمج يناسب شاشات اللابتوب الصغيرة والكبيرة.

## 🚀 تقنيات المشروع (Tech Stack)
- **Python 3.13**
- **CustomTkinter** (لواجهة المستخدم الحديثة)
- **Git Engine** (للمزامنة وإدارة النسخ)
- **JSON DB** (لتخزين البيانات محلياً بشكل سريع)

## 📦 طريقة التشغيل (Setup)
إذا كنت تريد تشغيل الكود من المصدر، تأكد من تثبيت المكتبات اللازمة:
```bash
pip install customtkinter
python AmerSync.py
```
## 🛠️ التحويل إلى EXE
لتحويل المشروع إلى ملف تنفيذي مستقل مع الأيقونة الكريستالية:
```bash
pyinstaller --noconsole --onefile --collect-all customtkinter --icon="icon.ico" --name "AmerSync_Crystal" AmerSync.py
```
